# Food Express React

## Run

```bash
npm install
npm run dev:All
```

Then open the local URL shown by Vite.

## Files

- `src/App.jsx` — React component
- `src/style.css` — converted CSS
- `src/main.jsx` — React entry point
- `src/assets/` — local images/videos
